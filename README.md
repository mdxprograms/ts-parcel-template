# TypeScript + parcel starter template

### Getting started

1. Use [degit](https://github.com/Rich-Harris/degit) to create new project directory

`degit https://github.com/mdxprograms/ts-parcel-template <your_app_name>`

2. `nvm use` or `nvm i`

3. `npm start`
